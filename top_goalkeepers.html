Scoring criteria for goalkeepers :

  • For each goal scored : 6 points
  • For a clean sheet : 4 points
  • For every 3 shot save : 1 point
  • For every 2 goals conceded : (-1) points
  • For playing up to 60 minutes : 1 point
  • For playing 60 minutes or more : 2 points
  • For each goal assist : 3 points
  • Bonus points for the best 3 players : 1-3 points
  • For each penalty save : 5 points
  • For each penalty miss : (-2) points
  • For each yellow card : (-1) points
  • For each red card : (-3) points
  • For each own goal : (-2) points
